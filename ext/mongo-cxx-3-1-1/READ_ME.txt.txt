If you compile in debug :
	-Delete the include directory
	-Delete the lib directory
	-Go into the debugVersion directory
	-Copy the include and lib directory
	-Paste it in parent dir. (mongo-cxx-3-1-1)

If you compile in release :
	-Delete the include directory
	-Delete the lib directory
	-Go into the releaseVersion directory
	-Copy the include and lib directory
	-Paste it in parent dir. (mongo-cxx-3-1-1)
